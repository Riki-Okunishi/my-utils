#### エイリアス

+ `ns3()`
  + `./waf` のエイリアス
    + `./waf condifure` と `./waf build` はそのまま `ns3` に置換できる
+ `suns3()`
  + `sudo ./waf` のエイリアス
    + 使い方は `ns3()` と同じ
    + **現状パスワード確認をされないので注意**
      + 予め何かのコマンドを `sudo` で実行しておく必要がある．
